# 甜筒页面飘屏弹幕被导航栏遮挡 - 修复计划

## 问题根因分析

### 1. 当前 z-index 层级关系

| 组件 | z-index | Position | 文件位置 |
|------|---------|----------|----------|
| 导航栏 Header | `z-40` | fixed | TiantongPage.tsx:L236 |
| 飘屏弹幕 HorizontalDanmaku | `z-40` | fixed | HorizontalDanmaku.tsx:L64 |

### 2. 问题原因

**甜筒页面**的飘屏弹幕与导航栏使用了**相同的 z-index (z-40)**，且都使用 `fixed` 定位。由于弹幕组件在 DOM 中位于导航栏之后，根据 CSS 层叠规则，后渲染的元素会覆盖前面的元素，导致弹幕显示在导航栏之上，遮挡了导航栏内容。

### 3. 与其他页面对比

| 页面 | 弹幕 z-index | 导航栏 z-index | 处理方式 |
|------|-------------|---------------|----------|
| **余小C** | `z-30` | `z-40` | 弹幕层级低于导航栏，并设置 `top: "80px"` 避开导航栏 |
| **驴酱** | `z-31` | `z-40` | 弹幕层级略低于导航栏 |
| **甜筒** | `z-40` | `z-40` | ❌ 同级，导致遮挡问题 |

## 修复方案

### 方案一：降低弹幕 z-index（推荐）

参考余小C页面的实现，将弹幕的 z-index 从 `z-40` 降低到 `z-30`，并添加 `top` 偏移避开导航栏区域。

**修改文件**: `frontend/src/features/tiantong/components/HorizontalDanmaku.tsx`

**具体修改**:
```tsx
// 修改前 (第64行)
<div className="fixed inset-0 pointer-events-none z-40 overflow-hidden">

// 修改后
<div 
  className="fixed inset-0 pointer-events-none z-30 overflow-hidden"
  style={{
    top: "80px", // 避开导航栏
    height: "calc(100vh - 80px)", // 占据剩余视口高度
  }}
>
```

**优点**:
- 与余小C页面保持一致，便于维护
- 弹幕不会遮挡导航栏
- 导航栏保持 `z-40`，确保在其他内容之上

### 方案二：提高导航栏 z-index

将导航栏的 z-index 提高到 `z-50`，使其高于弹幕的 `z-40`。

**修改文件**: `frontend/src/features/tiantong/TiantongPage.tsx`

**具体修改**:
```tsx
// 修改前 (第236行)
<header className="fixed top-0 left-0 right-0 z-40 w-full ...">

// 修改后
<header className="fixed top-0 left-0 right-0 z-50 w-full ...">
```

**缺点**:
- 需要同步修改 LIVE 标签的 z-index（当前也是 z-50）
- 可能与其他弹窗组件（VideoModal 也是 z-50）产生冲突

### 方案三：调整 DOM 顺序

将弹幕组件移动到导航栏之前渲染，利用 DOM 顺序控制层级。

**缺点**:
- 改变了组件结构，可能影响其他样式
- 不符合语义化 HTML 结构

## 推荐方案：方案一

选择方案一，理由如下：
1. **一致性**: 与项目中其他页面（余小C、驴酱）的实现保持一致
2. **低风险**: 只修改弹幕组件，不影响导航栏和其他组件
3. **可维护性**: 代码结构清晰，便于后续维护
4. **用户体验**: 弹幕从导航栏下方开始飘动，不影响导航栏交互

## 实施步骤

1. **修改弹幕组件**:
   - 文件: `frontend/src/features/tiantong/components/HorizontalDanmaku.tsx`
   - 将 `z-40` 改为 `z-30`
   - 添加 `style` 属性设置 `top: "80px"` 和 `height: "calc(100vh - 80px)"`

2. **验证修复**:
   - 启动开发服务器
   - 检查弹幕是否从导航栏下方开始飘动
   - 确认导航栏内容不被遮挡
   - 测试滚动页面时弹幕行为是否正常

3. **同步更新测试**:
   - 根据用户规则，修改代码后同步更新对应的测试代码
   - 检查是否有相关的单元测试或 E2E 测试需要更新
