# 时间轴展示优化 - 交互设计规格

## 一、视图切换组件规格

### 1.1 组件结构

```
┌────────────────────────────────────────────────────────────┐
│                      VideoToolbar                          │
├────────────────────────────────────────────────────────────┤
│  [时光轴] [网格] [列表]   │  [筛选▼] [排序▼]  [🔍搜索框]   │
└────────────────────────────────────────────────────────────┘
```

### 1.2 视图切换按钮组

| 元素 | 规格 |
|------|------|
| 按钮样式 | 图标 + 文字，64px × 36px |
| 激活状态 | 背景色填充，字体加粗 |
| 非激活状态 | 透明背景，灰色文字 |
| 过渡动画 | 150ms ease-out |
| 图标 | 时光轴📅 网格🔢 列表📋 |

**交互行为：**
- 点击切换视图，保留当前筛选条件
- 切换时滚动位置重置到顶部
- 激活按钮显示当前视图模式

### 1.3 筛选下拉菜单

| 维度 | 选项值 |
|------|--------|
| 时长 | `all`, `short` (0-5min), `medium` (5-30min), `long` (30min+) |
| 时间 | `all`, `week` (最近一周), `month` (最近一月), `year` (最近一年) |

**UI状态：**
- 默认显示"筛选"
- 选中后显示具体条件，如"筛选: 0-5分钟"

### 1.4 排序下拉菜单

| 维度 | 选项值 |
|------|--------|
| 排序方式 | `newest` (最新发布), `oldest` (最早发布), `popular` (最多播放) |

## 二、三种视图模式详细规格

### 2.1 时光轴模式（Timeline）

**当前实现，保持不变：**
- 左右交替布局
- 桌面端：视频卡片在中心线两侧交替
- 移动端：单列垂直布局

### 2.2 网格模式（Grid）

**CSS Grid 布局：**

```css
/* 桌面端 */
.video-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);  /* >1200px */
  grid-template-columns: repeat(3, 1fr);  /* >768px */
  gap: 16px;
}

/* 移动端 */
@media (max-width: 768px) {
  grid-template-columns: repeat(2, 1fr);
  gap: 12px;
}
```

**视频卡片（网格模式）：**
| 元素 | 规格 |
|------|------|
| 封面比例 | 16:9 |
| 封面圆角 | 8px |
| 标题 | 最多2行，溢出省略 |
| 标题字号 | 14px |
| 元信息 | UP主 + 时长，12px，灰色 |
| 悬停效果 | 轻微放大(1.02) + 阴影增强 |

### 2.3 列表模式（List）

**桌面端布局：**
```css
.video-list-item {
  display: flex;
  height: 120px;
  gap: 16px;
}

.video-list-item .cover {
  width: 180px;
  flex-shrink: 0;
}

.video-list-item .info {
  flex: 1;
  display: flex;
  flex-direction: column;
  justify-content: center;
}
```

**信息展示：**
| 元素 | 规格 |
|------|------|
| 标题 | 最多1行，溢出省略，16px |
| UP主 | 14px，带图标 |
| 时长 | 14px，右对齐 |
| 播放量 | 12px，灰色 |
| 发布日期 | 12px，灰色 |
| 简介 | 最多2行，14px，灰色 |

**移动端简化布局：**
- 封面宽度：120px
- 行高：100px

## 三、筛选交互规格

### 3.1 筛选面板

**触发方式：** 点击"筛选"按钮展开下拉面板

**面板内容：**
```
┌────────────────────────┐
│  时长                  │
│  [●] 全部              │
│  [ ] 0-5分钟          │
│  [ ] 5-30分钟          │
│  [ ] 30分钟以上        │
├────────────────────────┤
│  发布时间              │
│  [●] 全部              │
│  [ ] 最近一周          │
│  [ ] 最近一月          │
│  [ ] 最近一年          │
├────────────────────────┤
│        [重置] [应用]   │
└────────────────────────┘
```

### 3.2 筛选状态管理

**状态存储：**
```typescript
interface FilterState {
  duration: 'all' | 'short' | 'medium' | 'long';
  timeRange: 'all' | 'week' | 'month' | 'year';
  sortBy: 'newest' | 'oldest' | 'popular';
  searchQuery: string;
}
```

**URL同步：**
- 筛选条件同步到URL查询参数
- 支持分享筛选后的页面
- 示例：`/tiantong?view=grid&duration=short&sortBy=newest`

## 五、过渡动画规格

### 5.1 视图切换动画

| 场景 | 动画 | 时长 | 缓动函数 |
|------|------|------|----------|
| 视图切换 | 淡入淡出 | 300ms | ease-in-out |
| 卡片出现 | 依次淡入 | 50ms延迟/个 | ease-out |

### 5.2 筛选反馈动画

| 场景 | 动画 | 时长 |
|------|------|------|
| 筛选条件生效 | 结果区域淡入 | 200ms |
| 筛选结果数量变化 | 数字滚动动画 | 300ms |
