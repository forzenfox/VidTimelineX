## 三、技术架构设计

### 3.1 组件结构

```
src/
├── components/
│   └── video-view/              # 新增视图组件目录
│       ├── ViewSwitcher.tsx     # 视图切换器
│       ├── VideoGrid.tsx        # 网格视图
│       ├── VideoList.tsx        # 列表视图
│       ├── FilterPanel.tsx      # 筛选面板
│       ├── SortDropdown.tsx     # 排序下拉
│       └── index.ts             # 导出入口
├── features/
│   ├── tiantong/
│   │   └── TiantongPage.tsx    # 集成视图组件
│   ├── lvjiang/
│   │   └── LvjiangPage.tsx
│   └── yuxiaoc/
│       └── YuxiaocPage.tsx
└── hooks/
    ├── useVideoFilter.ts        # 筛选逻辑hook
    └── useViewPreference.ts      # 用户偏好hook
```

### 3.2 核心数据流

```
用户操作（切换视图/筛选/排序）
    ↓
更新FilterState状态
    ↓
计算筛选后的视频列表（useMemo）
    ↓
根据viewMode渲染对应组件
    ↓
更新URL参数（可选）
    ↓
保存到localStorage（偏好记忆）
```

### 3.3 类型定义

```typescript
// 视图模式
type ViewMode = 'timeline' | 'grid' | 'list';

// 筛选状态
interface FilterState {
  viewMode: ViewMode;
  duration: 'all' | 'short' | 'medium' | 'long';
  timeRange: 'all' | 'week' | 'month' | 'year';
  sortBy: 'newest' | 'oldest' | 'popular';
  searchQuery: string;
};

// 视频卡片尺寸
type CardSize = 'small' | 'medium' | 'large';
```

## 六、风险识别与应对

| 风险 | 影响 | 应对措施 |
