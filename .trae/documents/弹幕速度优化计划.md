# 三个页面水平飘屏弹幕速度优化计划

**创建时间：** 2026-02-28
**基准：** yuxiaoc 页面速度

---

## 📊 当前速度设置对比

| 页面 | duration 范围 | delay | 说明 |
|------|---------------|-------|------|
| **yuxiaoc** | 8-12 秒 (桌面端) / 10-14 秒 (移动端) | 0.3 秒 | 基准速度 |
| **甜筒** | 6-10 秒 | 0.2 秒 | 比 yuxiaoc 快 |
| **驴酱** | 6-10 秒 | 0.2 秒 | 比 yuxiaoc 快 |

### 详细分析

**yuxiaoc 页面 (基准)**：
```tsx
// duration: 8-12 秒 (桌面端)
map.set(i, (isMobile ? 10 : 8) + pseudoRandom * 4);
// delay: 0.3 秒
delay: i * 0.3,
```

**甜筒页面 (当前)**：
```tsx
// duration: 6-10 秒
map.set(i, 6 + pseudoRandom * 4);
// delay: 0.2 秒
delay: i * 0.2,
```

**驴酱页面 (当前)**：
```tsx
// duration: 6-10 秒
map.set(i, 6 + pseudoRandom * 4);
// delay: 0.2 秒
delay: i * 0.2,
```

---

## 🎯 优化目标

以 yuxiaoc 页面为基准，统一三个页面的弹幕速度：
- **duration**: 8-12 秒 (桌面端)
- **delay**: 0.3 秒

delay: i * 0.2,

// 修改后
