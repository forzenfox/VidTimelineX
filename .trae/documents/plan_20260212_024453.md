**修改位置**：`Header.tsx` 第 145 行

```tsx
// 修改前
{navItems.map((item) => {
  // 渲染逻辑
})}

// 修改后
{navItems.filter(item => item.id !== "danmaku").map((item) => {
  // 渲染逻辑
})}
```

这样修改只影响 Navigation - Desktop 的
