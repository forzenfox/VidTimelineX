## 修复方案

**问题**：`yuxiaoc.css` 使用了 Tailwind v3 语法，但项目使用 Tailwind v4，导致响应式类（如 `lg:flex`）无法生效。

**修复步骤**：

1. **修改 `yuxiaoc.css` 文件**：将 Tailwind v3 语法
   ```css
   @tailwind base;
   @tailwind components;
   @tailwind utilities;
   ```
   替换为 Tailwind v4 语法：
   ```css
   @import "tailwindcss";
   ```

2. **验证修复**：重新加载页面，确认 `externalLinks` 区域在桌面端正确显示

**影响范围**：仅修改 `yuxiaoc.css` 文件的前 3 行
