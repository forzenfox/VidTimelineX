## 📋 问题概述

根据 Playwright 分析结果，需要修复以下问题：

1. **问题 1（高优先级）**：甜筒页面切换主题后飘屏弹幕不触发
2. **问题 2（中优先级）**：驴酱页面飘屏弹幕文字不明显
3. **问题 3（低优先级）**：yuxiaoc 页面 - 无需修复，功能正常

***

## 🎯 修复目标

### 问题 1：甜筒页面切换主题后飘屏弹幕不触发

* **目标**：主题切换时，飘屏弹幕应重新生成并触发

* **验收标准**：

  * 切换主题后，飘屏弹幕立即重新生成

  * 弹幕颜色与新主题匹配

  * 弹幕动画流畅，无卡顿

  * Playwright E2E 测试通过

### 问题 2：驴酱页面飘屏弹幕文字不明显

* **目标**：增强弹幕文字可见性，提高对比度

* **验收标准**：

  * 文字在任何背景下都清晰可见

  * 字体大小适中（20-22px）

  * 文字阴影增强，有白色描边效果

  * 添加背景色和边框增强

  * Playwright 视觉回归测试通过

## 📅 实施步骤

### 阶段 1：修复甜筒页面主题切换问题（预计：0.5 天）

#### 任务 1.1：修改 TiantongPage.tsx

**文件**：`frontend/src/features/tiantong/TiantongPage.tsx`

**步骤**：

1. 添加 `danmakuKey` 状态
2. 修改 `toggleTheme` 函数，增加 key 更新逻辑
3. 在 `<HorizontalDanmaku>` 组件上使用 `key` prop

**代码变更**：

```typescript
// 添加状态
const [danmakuKey, setDanmakuKey] = useState(0);

// 修改 toggleTheme
const toggleTheme = useCallback((newTheme: "tiger" | "sweet") => {
  setTheme(newTheme);
  setDanmakuKey(prev => prev + 1); // 强制重新渲染
  document.documentElement.setAttribute("data-theme", newTheme);
}, []);

// 使用 key
<HorizontalDanmaku key={danmakuKey} theme={theme} />
```

### 阶段 2：修复驴酱页面弹幕文字问题（预计：0.5 天）

#### 任务 2.1：修改 HorizontalDanmaku.tsx

**文件**：`frontend/src/features/lvjiang/components/HorizontalDanmaku.tsx`

**步骤**：

1. 修改弹幕样式，增强文字可见性
2. 增加字体大小
3. 增强文字阴影效果
4. 添加背景色和边框

**代码变更**：

```typescript
style={{
  top: `${item.top * 100}%`,
  left: 0,
  transform: "translateX(100vw)",
  opacity: 1,
  animation: `lvjiang-danmaku-horizontal ${item.duration}ms linear ${item.delay}ms forwards`,
  fontSize: theme === "dongzhu" ? "20px" : "22px", // 增大字体
  color: theme === "dongzhu" ? "#2E86C1" : "#D68910", // 更深的颜色
  textShadow:
    theme === "dongzhu"
      ? "0 0 10px rgba(52, 152, 219, 0.8), 0 0 20px rgba(52, 152, 219, 0.6), 2px 2px 4px rgba(0,0,0,0.9), -1px -1px 0 #fff, 1px -1px 0 #fff, -1px 1px 0 #fff, 1px 1px 0 #fff"
      : "0 0 10px rgba(243, 156, 18, 0.8), 2px 2px 4px rgba(0,0,0,0.9), -1px -1px 0 #fff, 1px -1px 0 #fff, -1px 1px 0 #fff, 1px 1px 0 #fff",
  background: `linear-gradient(180deg, ${theme === "dongzhu" ? "rgba(52, 152, 219, 0.15)" : "rgba(243, 156, 18, 0.15)"} 0%, transparent 100%)`,
  border: `1px solid ${theme === "dongzhu" ? "rgba(52, 152, 219, 0.5)" : "rgba(243, 156, 18, 0.5)"}`,
  fontWeight: "900", // 加粗
  letterSpacing: "1.5px",
  willChange: "transform, opacity",
  backfaceVisibility: "hidden",
  perspective: 1000,
}}
```
