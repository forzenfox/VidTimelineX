# 列表模式封面尺寸优化计划

## 项目概述

在列表展示模式下，对视频封面图片进行尺寸优化，适当增加其显示比例以提升用户视觉体验。当前 compact 尺寸使用 96x128px (w-24 h-32) 的固定尺寸，需要适当放大以提升视觉效果，同时确保性能和响应式设计。

## 设计规范

### 封面尺寸优化

| 尺寸类型          | 优化前                  | 优化后                           | 变化      |
| ------------- | -------------------- | ----------------------------- | ------- |
| compact       | w-24 h-32 (96x128px) | w-32 h-48 (128x192px)         | +33% 宽度 |
| compact (响应式) | 固定尺寸                 | 移动端: w-28 h-42, 桌面: w-32 h-48 | 响应式调整   |

### 性能优化策略

* **图片懒加载**: 保持现有懒加载策略

* **图片尺寸**: 确保使用适当尺寸的封面图片

* **缓存**: 利用浏览器缓存

### 响应式断点

| 断点              | 封面尺寸                  |
| --------------- | --------------------- |
| < 640px (移动端)   | w-28 h-42 (112x168px) |
| ≥ 640px (平板/桌面) | w-32 h-48 (128x192px) |
