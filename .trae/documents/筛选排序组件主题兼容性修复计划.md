# 筛选排序组件主题兼容性修复计划

## 项目概述

修复甜筒页面 tiger 主题和驴酱页面 kaige 主题下 FilterDropdown 和 SortDropdown 组件文字看不清的问题。问题根源是这两个组件使用了硬编码的颜色值，而非主题 CSS 变量。

***

## 问题分析

### 受影响的页面和主题

| 页面       | 主题           | 背景色       | 文字可见性 |
| -------- | ------------ | --------- | ----- |
| tiantong | tiger (深色)   | `#2c3e50` | ❌ 看不清 |
| tiantong | sweet (亮色)   | `#fff8dc` | ✅ 正常  |
| lvjiang  | dongzhu (亮色) | `#FFFEF7` | ✅ 正常  |
| lvjiang  | kaige (深色)   | `#1A1A2E` | ❌ 看不清 |

### 根本原因

FilterDropdown 和 SortDropdown 组件使用硬编码颜色：

* `text-gray-700` - 在深色背景下看不清

* `border-gray-300` - 边框不明显

* `bg-white` - 背景不透明，与主题不协调

* `bg-gray-100` / `hover:bg-gray-100` - 悬停效果不协调

## 设计规范

### 需要替换的硬编码颜色映射

| 原硬编码值               | 替换为主题变量                       | 用途        |
| ------------------- | ----------------------------- | --------- |
| `text-gray-700`     | `text-foreground`             | 主要文字      |
| `text-gray-900`     | `text-foreground`             | 选中项文字     |
| `border-gray-300`   | `border-border`               | 边框        |
| `border-gray-200`   | `border-border/50`            | 分隔线       |
| `bg-white`          | `bg-card/95 backdrop-blur-md` | 背景色（玻璃拟态） |
| `bg-gray-100`       | `bg-muted/50`                 | 悬停背景      |
| `bg-gray-50`        | `bg-muted/30`                 | 悬停背景      |
| `hover:bg-gray-100` | `hover:bg-muted/50`           | 悬停效果      |
| `bg-transparent`    | `bg-card/60 backdrop-blur-md` | 触发按钮背景    |

### 玻璃拟态效果

* 添加 `backdrop-blur-md` 实现毛玻璃效果

* 使用 `bg-card/60` \~ `bg-card/95` 半透明背景

* 与 ViewSwitcher 和 VideoViewToolbar 保持一致
