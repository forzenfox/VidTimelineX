# 搜索框问题修复计划

## 问题分析

### 问题1: 进行搜索后无法重置(恢复显示全部结果)

**根本原因**:
- `SearchButton` 组件是**无状态组件**，只负责触发搜索，不显示当前搜索状态
- 搜索后输入框会被清空 (`setQuery("")`)，用户无法看到当前正在搜索的关键词
- 没有提供"清除搜索"功能，用户无法快速重置搜索

**影响范围**:
- 甜筒页面 (TiantongPage)
- 驴酱页面 (LvjiangPage)
- 鱼小C页面 (CanteenHall)

### 问题2: 进行搜索时UI排版异常

**根本原因**:
- `VideoViewToolbar` 中使用了 `variant="expanded"` 的 SearchButton
- expanded 模式的搜索框宽度固定为 `w-80`，可能与布局不协调
- 搜索框、视图切换按钮、筛选/排序按钮的间距和对齐方式需要优化

**截图分析**:
- 搜索框显示"C"但搜索状态显示"余小C"，说明输入框和实际搜索状态不同步
- 搜索框和右侧按钮的间距可能不均匀

## 修复方案

### 修复1: 添加搜索状态显示和清除功能

#### 1.1 修改 SearchButton 组件
**文件**: `frontend/src/components/video-view/SearchButton.tsx`

**修改内容**:
1. 添加 `currentQuery` prop 用于显示当前搜索关键词
2. 添加 `onClear` prop 用于清除搜索
3. 当存在当前搜索词时，显示搜索状态标签和清除按钮
4. 保持输入框与当前搜索状态同步

#### 1.2 修改 VideoViewToolbar 组件
**文件**: `frontend/src/components/video-view/VideoViewToolbar.tsx`

**修改内容**:
1. 传递 `searchQuery` 给 SearchButton 作为 `currentQuery`
2. 添加 `onClearSearch` 回调用于清除搜索

#### 1.3 修改 IconToolbar 组件
**文件**: `frontend/src/components/video-view/IconToolbar.tsx`

**修改内容**:
1. 传递 `searchQuery` 给 SearchButton 作为 `currentQuery`
2. 添加 `onClearSearch` 回调用于清除搜索
