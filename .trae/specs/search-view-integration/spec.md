# 搜索与视图切换整合优化 - 设计规范文档

## 变更ID
search-view-integration

## 变更目标

### 问题陈述
- **当前状态**：甜筒页面的视频搜索框与视图切换区域相互独立，造成视觉分散；驴酱页面完全缺失搜索功能
- **优化目标**：将视频搜索框整合到视图切换区域中，实现功能与视觉的统一
- **适用范围**：同时覆盖甜筒页面和驴酱页面，确保功能一致性

## 动画与过渡规范

### 过渡时长
- **快速**: 150ms (悬停、聚焦)
- **标准**: 200ms (状态切换)
- **慢速**: 300ms (展开/收起)

### 缓动函数
- **默认**: ease-in-out
- **进入**: cubic-bezier(0.4, 0, 0.2, 1)
- **退出**: cubic-bezier(0.4, 0, 1, 1)
- **弹性**: cubic-bezier(0.68, -0.55, 0.265, 1.55)

### 搜索框展开动画
- **触发**: 点击搜索按钮
- **展开**: width 0 → 200px, 200ms ease-out
- **收起**: width 200px → 0, 150ms ease-in
- **图标切换**: 搜索图标 → 关闭图标, 150ms

### 视图切换动画
- **图标切换**: opacity + scale, 200ms
- **内容切换**: fade-in, 200ms
- **按钮反馈**: scale 0.95 on press, 100ms

### 下拉菜单动画
- **展开**: fade-in + slide-down, 200ms
- **收起**: fade-out, 150ms

## 交互设计规范

### 搜索功能交互
1. **点击搜索按钮**: 展开搜索输入框，自动聚焦
2. **输入内容**: 实时显示清除按钮，300ms防抖触发搜索
3. **点击清除**: 清空输入，保持聚焦
4. **按Enter**: 触发搜索，收起搜索框
5. **点击外部/按Esc**: 收起搜索框
6. **显示建议**: 匹配视频标题，最多5条（可选）

### 视图切换交互
1. **点击视图按钮**: 切换到下一个视图模式
2. **切换顺序**: timeline → grid → list → timeline
3. **图标变化**: 按钮图标立即更新为当前视图图标
4. **保存偏好**: 自动保存到localStorage
5. **Tooltip**: hover显示当前视图名称

### 排序/筛选交互
1. **点击按钮**: 展开下拉菜单
2. **选择选项**: 即时应用，关闭菜单
3. **有激活状态**: 显示指示点或高亮

## 无障碍规范

### 键盘导航
- **Tab**: 在按钮间切换
- **Enter/Space**: 激活按钮
- **Esc**: 关闭搜索框/下拉菜单
- **搜索框展开时**: 支持输入和清除

### ARIA属性
- 搜索按钮: `aria-label="搜索视频"`, `aria-expanded` (展开状态)
- 视图切换: `aria-label="切换视图，当前：时间线"`
- 排序按钮: `aria-label="排序"`, `aria-haspopup="listbox"`
- 筛选按钮: `aria-label="筛选"`, `aria-haspopup="listbox"`

### 对比度要求
- 文字与背景对比度 ≥ 4.5:1
- 交互元素对比度 ≥ 3:1
