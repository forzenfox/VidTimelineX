## 4. 全局通用配色

### 4.1 按钮配色

| 状态 | 背景色 | 文本色 | 边框色 | 阴影色 |
|------|--------|--------|--------|--------|
| 正常 | 渐变或主题色 | 白色或主题文本色 | 主题色或透明 | 主题色半透明 |
| 悬停 | 主题色变亮 | 白色或主题文本色 | 主题色 | 主题色半透明加深 |
| 点击 | 主题色变暗 | 白色或主题文本色 | 主题色 | 主题色半透明 |
| 禁用 | 灰色 | 浅灰色 | 浅灰色 | 无 |

## 5. 配色使用规则

### 5.1 主题切换

- 使用 `data-theme` 属性切换主题：
  - `data-theme="tiger"`：使用 Tiger 主题配色
  - `data-theme="sweet"`：使用 Sweet 主题配色
  - `data-theme="dongzhu"`：使用洞主主题配色
  - `data-theme="kaige"`：使用凯哥主题配色

### 5.2 颜色变量使用

- 使用 CSS 变量引用颜色，避免硬编码：
  - `var(--background)`：背景色
  - `var(--foreground)`：文本色
  - `var(--primary)`：主色调
  - `var(--secondary)`：次色调
  - `var(--accent)`：强调色
  - `var(--card)`：卡片背景
  - `var(--border)`：边框色
  - `var(--shadow-color)`：阴影色

### 5.3 颜色对比度

- 确保文本与背景的对比度符合 WCAG AA 标准（至少 4.5:1）
- 确保交互元素的状态变化有足够的颜色对比度

## 6. 配色方案应用示例

### 6.1 甜筒页面应用

- **Tiger 主题**：深色背景，橙色强调，金属银色文本，营造威武、专业的氛围
- **Sweet 主题**：浅色背景，粉色强调，深灰色文本，营造可爱、温馨的氛围

### 6.2 驴酱页面应用

- **洞主主题**：天蓝色系，营造清新、活力的氛围
- **凯哥主题**：深色系，营造沉稳、专业的氛围
