# 「亿口甜筒」时光视频集 - 非商业素材和字体推荐

## 1. 字体推荐

### 1.1 中文免费字体

#### 思源黑体 (Noto Sans SC)
- **特点**：Google开发的开源中文字体，字形清晰，可读性强
- **适用场景**：正文、标题、导航栏
- **获取方式**：Google Fonts
- **引入代码**：
  ```html
  <link href="https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@300;400;500;700;900&display=swap" rel="stylesheet">
  ```

#### 思源宋体 (Noto Serif SC)
- **特点**：优雅的衬线字体，适合正式场合
- **适用场景**：引用、日期、特殊强调文本
- **获取方式**：Google Fonts
- **引入代码**：
  ```html
  <link href="https://fonts.googleapis.com/css2?family=Noto+Serif+SC:wght@400;600;700&display=swap" rel="stylesheet">
  ```

#### 站酷快乐体
- **特点**：活泼可爱，符合甜筒的形象
- **适用场景**：标题、强调文本、品牌名称
- **获取方式**：站酷字体库（免费非商用）
- **引入方式**：下载后本地引入

#### 站酷庆科黄油体
- **特点**：圆润可爱，有奶油质感
- **适用场景**：Sweet主题的标题和强调文本
- **获取方式**：站酷字体库（免费非商用）
- **引入方式**：下载后本地引入

### 1.2 英文免费字体

## 3. 实现建议

### 3.1 字体实现

1. **Google Fonts 集成**
   - 在 `index.html` 文件中添加字体链接
   - 在 `tailwind.config.js` 中配置字体

   ```javascript
   // tailwind.config.js
   module.exports = {
     theme: {
       extend: {
         fontFamily: {
           'sans': ['Noto Sans SC', 'sans-serif'],
           'serif': ['Noto Serif SC', 'serif'],
           'poppins': ['Poppins', 'sans-serif'],
           'fredoka': ['Fredoka One', 'cursive'],
         },
       },
     },
   }
   ```

2. **本地字体集成**
   - 创建 `src/fonts` 目录，存放下载的字体文件
   - 在 `index.css` 中添加 `@font-face` 规则

   ```css
   /* 站酷快乐体 */
   @font-face {
     font-family: 'ZCOOLKuaiLe';
     src: url('./fonts/ZCOOLKuaiLe-Regular.ttf') format('truetype');
     font-weight: normal;
     font-style: normal;
     font-display: swap;
   }

   /* 站酷庆科黄油体 */
   @font-face {
     font-family: 'ZCOOLQingKeHuangYou';
     src: url('./fonts/ZCOOLQingKeHuangYou-Regular.ttf') format('truetype');
     font-weight: normal;
     font-style: normal;
     font-display: swap;
   }
   ```

### 3.2 素材实现

1. **图片优化**
   - 使用适当的图片格式（JPG、PNG、WebP）
   - 实现图片懒加载
   - 为不同屏幕尺寸提供不同分辨率的图片

2. **图标使用**
   - 优先使用SVG图标，确保清晰度
   - 对于Lucide React图标，按需导入以减少包大小

3. **背景实现**
   - 使用CSS变量管理背景颜色和渐变
   - 为不同主题设置不同的背景样式

### 3.3 性能优化

1. **字体加载优化**
   - 使用 `font-display: swap` 确保文本快速显示
   - 考虑使用字体子集，只包含常用字符

2. **素材加载优化**
   - 压缩图片和SVG
   - 使用CDN加载外部资源
   - 实现资源预加载
