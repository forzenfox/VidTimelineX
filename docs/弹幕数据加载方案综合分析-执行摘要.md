   - 浪费率：92%

## 🎯 最优方案

### **方案 1：侧边栏弹幕 - 运行时动态组合**

**数据流：**

```
danmaku.json（文本） ─┐
                      ├→ 运行时组合 → DanmakuMessage → React 组件
users.json（用户） ───┘
```

**核心代码：**

```typescript
import { DanmakuGenerator } from '@/shared/danmaku';
import danmakuData from "./danmaku.json";
import usersData from "./users.json";

const generator = new DanmakuGenerator({
  theme: 'blood',
  textPool: danmakuData.bloodDanmaku,
  users: usersData,
});

const messages = generator.generateBatch({
  count: 12,
  includeUser: true,  // ✅ 包含用户信息
});
```

**优势：**
- ✅ 开发体验优秀（修改即生效）
- ✅ 灵活性极强（支持实时更新）
- ✅ 数据体积小（~6KB）
- ✅ 性能足够（<5ms/条）

---

### **方案 2：飘屏弹幕 - 简单文本数组**

**数据流：**

```
danmaku.txt（纯文本） → 组件直接使用 → CSS 动画渲染
```

**核心代码：**

```typescript
import danmakuText from "./danmaku.txt?raw";
const textPool = danmakuText.split('\n').filter(line => line.trim());

const items = textPool.map((text, i) => ({
  id: `danmaku-${i}`,
  text,  // ✅ 直接使用文本
  top: (i % 8) * 10,
  delay: i * 0.3,
  duration: 8 + Math.random() * 4,
}));
```

**优势：**
- ✅ 数据体积极小（3KB vs 25KB，节省 88%）
- ✅ 无需构建步骤
- ✅ 修改即生效
- ✅ 性能差异可忽略（+8ms，人眼无法感知）
