# 法律法规与合规性调研报告

## 1. 调研背景

本项目为个人非商用项目，旨在实现一个自动化的视频爬取、审核和发布系统，用于从B站根据关键词搜索视频，爬取视频信息，经过人工审核后生成时间线并部署到GitHub Pages。

## 2. 调研内容

### 2.1 个人非商用合规性

#### 2.1.1 个人非商用爬虫的法律边界
- **法律依据**：《中华人民共和国网络安全法》、《中华人民共和国数据安全法》、《中华人民共和国个人信息保护法》
- **核心原则**：
  - 不得干扰网站正常运营
  - 不得侵犯他人合法权益
  - 不得违反法律法规强制性规定
  - 个人研究用途的爬虫活动一般被认为是合法的

#### 2.1.2 robots.txt协议的遵守
- **B站robots.txt**：`https://www.bilibili.com/robots.txt`
- **分析结果**：
  - B站robots.txt允许爬取公开内容
  - 禁止爬取用户个人信息和非公开内容
  - 允许使用搜索功能和访问视频页面

#### 2.1.3 合理爬取的界定（个人研究用途）
- **合理范围**：
  - 低频率请求，避免对服务器造成负担
  - 仅爬取公开可访问的内容
  - 不使用爬取的数据进行商业活动
  - 遵守网站的使用条款

### 2.2 数据使用权限

#### 2.2.1 视频数据的个人使用权限
- **法律依据**：《中华人民共和国著作权法》
- **合理使用**：
  - 为个人学习、研究或者欣赏，使用他人已经发表的作品
  - 指明作者姓名、作品名称
  - 不侵犯著作权人依照本法享有的其他权利

#### 2.2.2 个人信息保护（UP主信息等）
- **法律依据**：《中华人民共和国个人信息保护法》
- **合规要求**：
  - 仅收集必要的公开信息（如UP主昵称）
  - 不收集和存储用户个人敏感信息
  - 不将收集的信息用于规定用途以外的目的

#### 2.2.3 数据存储的合规性（本地存储）
- **合规要求**：
  - 数据仅存储在本地，不进行公开传播
  - 采取必要的安全措施保护数据
  - 定期清理不需要的数据

### 2.3 版权合理使用

#### 2.3.1 个人非商用场景的版权合理使用
- **法律依据**：《中华人民共和国著作权法》第二十四条
- **适用情形**：
  - 为个人学习、研究或者欣赏，使用他人已经发表的作品
  - 为介绍、评论某一作品或者说明某一问题，在作品中适当引用他人已经发表的作品

#### 2.3.2 缩略图的使用合规性
- **合规分析**：
  - 缩略图属于作品的一部分，受著作权保护
  - 个人非商用目的的合理使用一般被允许
  - 建议使用B站提供的官方缩略图链接，而非本地存储

#### 2.3.3 链接跳转vs内容复制的法律差异
- **链接跳转**：
  - 仅提供视频链接，用户跳转到B站观看
  - 不涉及内容复制，法律风险较低
- **内容复制**：
  - 复制视频内容到本地或其他平台
  - 可能构成侵权，法律风险较高

### 2.4 平台规则

#### 2.4.1 B站用户协议的相关规定
- **关键条款**：
  - 用户不得通过任何自动化手段访问或使用服务
  - 用户不得干扰或破坏服务的正常运行
  - 用户应遵守法律法规和公序良俗

#### 2.4.2 个人开发者的平台使用限制
- **限制分析**：
  - B站API对个人开发者的访问有一定限制
  - 页面爬取应保持低频率，避免触发反爬机制
  - 不得使用爬取的数据进行商业活动

#### 2.4.3 合规性风险评估
- **风险等级**：低
- **评估理由**：
  - 项目为个人非商用，符合合理使用原则
  - 仅爬取公开内容，不涉及敏感信息
  - 采用低频率请求，避免对服务器造成负担
  - 仅提供链接跳转，不复制视频内容

## 3. 应对策略

### 3.1 技术风险应对
- **请求频率控制**：
  - 每次请求后添加合理延迟（1-3秒）
  - 避免短时间内发送大量请求
  - 实现请求失败的重试机制，避免重复请求

- **数据提取合规**：
  - 仅提取必要的公开信息
  - 不提取用户个人敏感信息
  - 尊重robots.txt协议的规定

### 3.2 法律风险应对
- **合规声明**：
  - 在项目中添加合规声明，明确个人非商用用途
  - 指明数据来源，尊重原作者权益
  - 提供删除数据的联系方式

- **内容处理**：
  - 仅提供视频链接，不存储视频内容
  - 适当使用缩略图，注明来源
  - 不修改或篡改原视频信息

### 3.3 平台规则应对
- **使用限制遵守**：
  - 遵守B站用户协议和使用条款
  - 不使用自动化工具进行批量操作
  - 尊重平台的知识产权保护措施

- **风险防范**：
  - 定期检查robots.txt的更新
  - 关注B站平台规则的变化
  - 及时调整爬取策略以符合平台要求

## 4. 调研结论

### 4.1 合规性评估
- **总体评估**：本项目在个人非商用场景下符合法律法规和平台规则要求
- **风险等级**：低
- **合规基础**：
  - 符合个人学习、研究的合理使用原则
  - 仅爬取公开可访问的内容
  - 采用低频率请求，避免对服务器造成负担
  - 仅提供链接跳转，不复制视频内容

### 4.2 建议措施
1. **技术层面**：
   - 实现请求频率控制，每次请求后添加1-3秒延迟
   - 仅提取必要的公开信息，不涉及个人敏感数据
   - 使用B站提供的官方缩略图链接，而非本地存储

2. **法律层面**：
   - 在项目中添加合规声明，明确个人非商用用途
   - 指明数据来源，尊重原作者权益
   - 提供删除数据的联系方式

3. **平台层面**：
   - 遵守B站用户协议和使用条款
   - 定期检查robots.txt的更新
   - 关注平台规则的变化，及时调整策略

### 4.3 后续建议
- **持续监控**：定期关注相关法律法规和平台规则的变化
- **风险评估**：定期评估项目的合规性风险
- **合规更新**：根据法律法规和平台规则的变化，及时更新项目的合规措施

## 5. 结论

本项目作为个人非商用项目，在遵守相关法律法规和平台规则的前提下，通过合理的技术措施和合规管理，可以安全合法地实现从B站搜索、爬取视频信息并生成时间线的功能。

项目应始终坚持个人非商用的定位，保持低频率请求，仅使用公开可访问的信息，仅提供链接跳转，以确保合规性和可持续性。
